## 팀확정 첫 미팅

- 기술 스택 : React + Spring Boot + JPA
- 오프라인 회의 일정 : 화, 목 4시(1/21부터)
- 기획 주제 : 1월 회의 후 주제 선정 예정

- 기능 회의 후 개발 전 할 일 : docker 환경 세팅, 컨벤션 문서화

- 팀원별 희망 사항

  - 곽나영 : 코드 퀄리티 유지

  - 유현진 : React + Spring Boot 기술 스택 희망
